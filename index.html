Sequências e Séries

Sequência de Fibonacci

Fₙ = Fₙ₋₁ + Fₙ₋₂

Definição recursiva

Implementação com Memoization 

from functools import lru_cache

@lru_cache(maxsize=None)
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

Série de Taylor

eˣ = Σ xⁿ/n!

Expansão para exponencial

Aproximação de eˣ 

def exp_taylor(x, n_terms=10):
    approximation = 0.0
    for n in range(n_terms):
        approximation += x**n / factorial(n)
    return approximation
